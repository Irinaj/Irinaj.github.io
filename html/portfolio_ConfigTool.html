Config Tool (https://flextecs.com/)

Configurable Tool is a desktop application used to develop custom data analysis tools with dynamic menus, layouts, and relationships between data grids based on XML configurations. This application cut off the time and manual work required by solutions architects' to develop such tools and has made their lives easier while working with audit data; now they can completely focus on data rather than the application’s UI.
It mainly has two parts:
I. Maintain/Generate
II. Load the form with the controls based on the XML provided so that auditors can analyze/audit data as required.

With the help of this tool, one can have any number of grids and viewers at the specified location to view/analyse data. Major features included are:
➢ Maintain parent/child relations between the grids.
➢ Data manipulation based on configuration rules defined in Excel template files.
➢ Adding notes and uploading documents.
➢ Criteria-based conditional formatting.
➢ Add menus, submenu etc.
